数据类型判断

js中的数据类型分为基本数据类型以及引用数据类型 基本数据类型包含：String Boolean Number Null Undefined Symbol Bigint 引用数据类型包括: object function Array Date…

判断数据类型的方法主要有以下几种

  • typeof

typeof可返回所有基本数据类型的类型以及function其他引用类型返回object 返回的值为一个string类型,

bull的返回值为object 这是js设计中的一个遗留问题 （因为js设计时运行于32位的机器上为了节省空间将bull机器码设计为全零 被判定为object）

缺点：无法区分除function以外的object

  • instanceof

​ instanceof能够判断实例是否属于某种类型 即它只能用于区分object类型数据

​ instanceof 也可判断一个实例是其父类型或祖先类型；

instanceof的实现原理为判断实例的__proto__是否出现在左边的prototype以及其父类和祖先的prototype

  • Object.prototype.toString.call()

    Object.proto.toString().call的原理是当调用时，取值内部的[[class]]属性，拼接为[object,’class’] 这样的字符串返回

  • Construtor

    construtor是原型上对象的属性，指向构造函数。实例对象也可根据原型链查找到construtor属性根据这点可用来判断类型；

    缺点： 无法判断undefined和null的属性，且construtor的值可变。
